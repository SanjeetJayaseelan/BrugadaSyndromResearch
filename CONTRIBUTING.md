# Contributing

This started as a solo independent research project, so there's no formal process yet. If you spot a bug in the feature-extraction heuristics or want to help with external validation on a second BrS cohort, please open an issue first to discuss.

## Code style

Plain PEP 8, no auto-formatter enforced yet. Keep functions small and prefer explicit loops over cleverness — this is meant to be readable by a cardiologist skimming the methodology, not just by another engineer.
